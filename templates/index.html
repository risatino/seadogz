OVERVIEW: An Interactive Comparison Data Report

The goal of this project was to compare basketball outcomes data (games stats & injury data) to captured athlete monitoring data and evaluate for trends. Data was compiled from the Catapult Sport API, team game statistics, and injury data from one NCAA Division I team. Data exploration was completed in Python Pandas with data visualization in Tableau and machine learning in Sci-Kit Learn. The graphs and data below show our exploration and analysis.
